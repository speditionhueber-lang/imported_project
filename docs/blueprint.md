# **App Name**: Firestore Starter Kit

## Core Features:

- Customer Management: CRUD operations for customer data with fields like name, email, and address.
- Job Scheduling: Create and manage jobs, linking them to customers and scheduling their execution.
- Invoice Generation: Generate invoices for jobs, including calculation of net total and VAT.
- Role-based Access Control: Secure access to data based on user roles (staff, admin).
- Prefix Search: Allows prefix-based searching for customer names.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) for trustworthiness and stability.
- Background color: Light blue (#E6EDFF) for a clean and professional look.
- Accent color: Purple (#9D4EDD) to highlight key interactive elements.
- Body and headline font: 'Inter' sans-serif for clear and modern readability.
- Use flat, minimalist icons to represent data and actions.
- Clean, structured layouts to facilitate easy navigation.
